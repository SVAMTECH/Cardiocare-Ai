CardioCare AI – Predictive Risk Assessment for Acute Myocardial Infarction

## 🧩 Project Overview

CardioCare AI is a machine learning-powered web application designed for the **predictive assessment of Acute Myocardial Infarction (AMI)** risk. The platform enables healthcare providers to analyze patient medical and lifestyle data for early detection of heart attack risks, promoting proactive intervention and personalized care. The application leverages the XGBoost algorithm for accurate and scalable risk prediction, and follows well-established software engineering and data science best practices.

***

## Architecture

The project architecture is modular and scalable, designed to support real-time clinical use and future integrations:

1. **Presentation Layer:** Flask web application with user-friendly forms for medical data entry and result visualization.
2. **Application Layer:** Machine Learning backend using XGBoost, providing efficient API-driven predictions.
3. **Data Layer:** Secure storage and management of structured patient data for analysis and auditability.

## Components Used

- **Flask:** Serves as the web server and API framework, handling application routing and HTTP requests.
- **XGBoost:** Used as the primary machine learning algorithm for accurate and robust risk prediction of Acute Myocardial Infarction.
- **Python:** Provides the backbone for application logic, data processing, and the machine learning pipeline.
- **HTML, CSS, JavaScript:** Implements a responsive and intuitive user interface for data entry and result visualization.
- **scikit-learn:** Facilitates data preprocessing, model evaluation, and auxiliary machine learning tasks.
- **pandas, NumPy:** Utilized for efficient data manipulation, analysis, and numerical computations.
- **User Authentication:** Enables secure access control for different user roles including patients, clinicians, and administrators.

## 🚀 Deployment Steps

1. **Clone Repository:**  
   ```bash
   git clone https://github.com/your-username/cardiocare-ai.git
   cd cardiocare-ai
   ```
2. **Install Dependencies:**  
   ```bash
   pip install -r requirements.txt
   ```
3. **Start Application:**  
   ```bash
   python app.py
   ```
4. **Access the Application:**  
   Visit [http://localhost:5000/](http://localhost:5000/) in your browser.
